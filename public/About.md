# RosterCast v2.0.0

## Description
This is the public read-only directory for the RosterCast application.

## Public Access
Public visitors can view the approved member directory and map. They cannot edit members, send email, restore backups, change settings, or access private logs.

## Admin Maintenance
Set `ADMIN_PASSWORD` in the environment before deploying if maintenance login is required. Do not store production passwords in `data/config.json`.
